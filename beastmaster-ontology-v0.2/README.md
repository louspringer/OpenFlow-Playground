# Beastmaster Ontology v0.2

**What changed from v0.1 → v0.2 (based on feedback):**

- Added **Process/Workflow/Step** modeling.
- Added **QualityGate/Validation** concepts.
- Added **Tool/Execution** for integration and runs.
- Enhanced **Metrics**: types, thresholds, relationships (correlates/dependsOn/aggregates).
- Strengthened **Traceability** with strength, confidence, reason, and types (direct/indirect/derived).
- Expanded **Constraint** model with operators and values.
- Updated **SHACL** to validate new structures.

See `ontology/beastmaster.ttl` and `ontology/beastmaster.shacl.ttl`.
