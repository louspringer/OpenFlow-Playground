# 🔍 Workflow Visualization GUI

## Overview

The **Workflow Visualization GUI** is a comprehensive Streamlit application that provides an intuitive interface for visualizing all workflow extraction artifacts generated by our tested components. It offers a navigation framework organized by tested components and artifact types, making it easy to explore and analyze workflow extraction results.

## 🚀 Quick Start

### Launch the GUI

```bash
# Option 1: Use the launcher script (recommended)
python launch_gui.py

# Option 2: Direct Streamlit launch
uv run streamlit run src/workflow_visualization_gui.py

# Option 3: Manual launch with custom settings
uv run streamlit run src/workflow_visualization_gui.py --server.port 8501 --server.address localhost
```

The GUI will automatically open in your default web browser at `http://localhost:8501`.

## 🏗️ Architecture

### Core Components

The GUI integrates with all 8 tested workflow extraction components:

| **UC-9** | Performance Optimization | Performance metrics, optimization recommendations |

## 🔍 Troubleshooting

### Common Issues

#### 1. **Streamlit Not Starting**
```bash
# Check dependencies
uv sync

# Verify Streamlit installation
uv run python -c "import streamlit; print(streamlit.__version__)"

# Check port availability
netstat -tulpn | grep :8501
```

#### 2. **Component Import Errors**
```bash
# Verify component availability
uv run python -c "from src.control_flow_analyzer import ControlFlowAnalyzer"

# Check file paths
ls -la src/*.py

# Update dependencies
uv sync --upgrade
```

#### 3. **Artifact Generation Failures**
```bash
# Check file permissions
ls -la generated_activity_models/

# Verify output directory
mkdir -p generated_activity_models

# Check disk space
df -h .
```

#### 4. **Performance Issues**
```bash
# Monitor system resources
htop

# Check memory usage
free -h

# Analyze bottlenecks
uv run python src/performance_optimizer.py
```

### Debug Mode
```bash
# Enable debug logging
export STREAMLIT_LOGGER_LEVEL=debug

# Run with verbose output
uv run streamlit run src/workflow_visualization_gui.py --logger.level=debug
```
