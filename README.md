# OpenFlow Playground

A playground for experimenting with AI agents, multi-agent coordination, and cloud deployment.

## Features

- Multi-agent coordination framework
- GKE deployment capabilities
- Cost monitoring and optimization
- LangChain/LangGraph integration
- Beast Mode Framework integration

## Quick Start

```bash
# Install dependencies
uv sync

# Run tests
make test

# Deploy to GKE
./scripts/deploy-kiro-agent-gke.sh
```

## Cost Monitoring

This project includes cost monitoring tools to track GCP usage and optimize spending.
